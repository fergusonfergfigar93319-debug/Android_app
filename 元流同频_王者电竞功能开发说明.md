# 元流同频 — 王者电竞功能开发说明

**版本**：V1.0  
**读者**：Android / Web / 后端 / 产品  
**目的**：在「**王者荣耀**（对局）+ **王者电竞**（赛事/观赛）」双主线下，**系统化**描述已落地能力、数据契约与迭代方向，避免各端只做零散文案而缺少 **可验收的功能闭环**。

**合规**：本应用为 **第三方玩家工具**，非腾讯官方；赛事数据、队标、解说引用等须遵守版权与平台规则，对外展示保持 **「讨论/观赛心得」** 心智，避免冒充官方赛程或实时比分源（实时数据以后端授权与合约为准）。

---

## 一、定位与信息架构

| 维度 | 王者荣耀（本体） | 王者电竞（赛事向） |
|------|------------------|---------------------|
| **用户心智** | 排位/巅峰/英雄、对局技巧、版本变动 | KPL/杯赛/观赛、BP、战队、解说、赛后复盘 |
| **主入口** | 版本速递频道、英雄绑定的 AI 搭子、攻略/招募帖 | 同一套底栏；**资讯频道 `gameName=王者电竞`**、**`focusScenario=王者电竞`** 的智能体、广场 **赛事/活动** 分区与标签 |
| **推荐匹配** | 分路、英雄池、对局风格 | `favoriteEsportsHint`、标签含 `KPL`/`王者电竞`、观赛偏好（建档见 `OnboardingConfig`） |

**与峡谷对局的边界**：用户可能 **只玩不观赛** 或 **只看不打**；产品上不强制二选一，但 **UI 与推荐** 应根据 `GameInterestStore` 勾选与画像字段 **降权无关内容**（例如仅勾选王者电竞时，资讯默认优先赛事线）。

## 三、后端 / 接口层建议（与《对接说明》协同）

1. **资讯** `GET /feed/news`：查询参数 **`game`** 支持 `王者荣耀` | `王者电竞`（与客户端 `gameName` 一致）；赛事线可增加 **`subChannel`**（可选）：`kpl` | `cup` | `misc`（具体枚举与运营约定）。
2. **画像扩展**：用户勾选「王者电竞」时，持久化 **`favoriteEsportsHint`**（短句）或结构化 **`esportsPreferences`**（主追联赛、主看位置、是否看二路等），便于推荐与 LLM 上下文。
3. **智能体**：`POST /ai/agent/chat` 在 **`focusScenario === '王者电竞'`** 时，System Prompt 侧重 **BP、龙团节奏、换线、教练轮换意图** 等（与 `DesignedAgentPresets` 中 `extraInstructions` 方向一致），与 **纯英雄教学** 区分。
4. **广场检索**：`GameCatalog.searchAliasBlob()` 已含 `KPL、世冠、挑战者杯` 等；服务端分词/标签建议 **同源维护** 一份 JSON，避免 App/Web 热词不一致。

## 四、Web 端对齐清单

- **游戏兴趣页**：仅两项，id 与 `FollowGameCatalog.options` 一致。
- **版本速递**：资讯 Tab 支持 **按 `王者电竞` 过滤**；热词含 KPL/观赛/BP 等（与 App `defaultHotQueries` + `GameCatalog` 对齐）。
- **AI 搭子**：`focusScenario` 含 **王者电竞**；成品模板或说明与 `DesignedAgentPresets` 中科班预设等价。
- **智能体聊天**：电竞向快捷句、与 Android 相同的 **focus 分支**（见 `AgentPersonaResolver` 逻辑摘要）。
- **峡谷广场**：`event` 分区、标签池含 **KPL / 王者电竞 / 观赛**；发帖引导与搜索别名一致。

## 五、迭代路线图（建议）

| 阶段 | 内容 | 说明 |
|------|------|------|
| **P0（当前）** | 频道 + 标签 + 智能体人设 + 广场种子 + 推荐字段 | 双端同构、可演示闭环 |
| **P1** | **赛程卡片**（官方或合作数据源）、**订阅提醒**（本地/Push）、赛后 **一键生成讨论帖** | 需后端或合规数据源；客户端可先做 **静态下周程 + 深链广场** |
| **P2** | **战队/选手关注**（非官方需抽象为「兴趣标签」）、**_clip 时间轴**（用户上传观赛笔记） | 社区向，注意肖像与商标 |
| **P3** | **实时比分**、**语音连麦观赛** | 强依赖版权与基础设施 |

---

## 六、验收要点（王者电竞专项）

- [ ] 未选王者电竞时，仍可看到混合流；**选中后** 资讯/推荐是否 **优先赛事相关**（若产品要求）。
- [ ] `focusScenario=王者电竞` 时，智能体回复 **不出现** 纯英雄连招占满篇幅（除非用户明确问英雄）；应能聊 **BP、节奏、观赛入门**。
- [ ] 广场搜索 **「KPL」「王者电竞」「观赛」** 能命中种子或线上帖；发帖标签推荐含 `GameCatalog.forumGlobalTagPool()` 中的电竞词。
- [ ] Web 与 Android **同一套** `gameIds`、`focusScenario` 枚举字符串。
